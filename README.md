# mit-ocw_hands-on-deep-learning
*In progress - currently up to transformer-based contextual embeddings; finishing transformers, LLMs, and generative AI later in 2026.*

Implementations I wrote while following [MIT 15.773 Hands-On Deep Learning (Spring 2024)](https://ocw.mit.edu/courses/15-773-hands-on-deep-learning-spring-2024/). Rather than running the course's provided notebooks, I rebuilt each one in TensorFlow - filling in the homework code blanks without looking at solutions, and writing the GloVe embeddings from scratch.

What's in the notebooks
- DNN for heart disease prediction
- DNN and CNN for clothing item image classification
- CNN for emotion image classification
- Bag-of-words (multi-hot vector) and standalone embedding (GloVe) models for music genre classification from song lyrics
- GloVe embeddings implemented from scratch

What I've learned so far

The foundations of neural networks - gradient descent, backpropagation, activation functions - followed by CNNs (convolutional filters, pooling layers), natural language representation with multi-hot vectors and standalone embeddings like GloVe, and the basics of contextual embeddings produced by transformers.
